在 koa 中使用

埋点上报

可手动上报基于 http 协议的模块间调用上报

reportMCall(service, method, path, http_code, code)

参数说明

参数名 必选 默认值 说明
service 是 调用的服务或模块名
method 是 调用请求的 http 方法(GET/POST)
path 是 调用请求的路径
http_code 是 调用请求的 http 码
code 否 0 模块返回的错误码

示例

其他

  • 访问 http://ip:port/prometheus 查看 metrics 数据
  • 访问 http://ip:port/swagger/ui 查看 swagger 可视化监控数据
  • promConf配置数据，参考https://swaggerstats.io/guide/conf.html
  • 使用非 koa 框架或有关swagger-stats的更多信息，参考https://swaggerstats.io/guide/
